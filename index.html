🍏 iOS · Orion Extension
  1. Install the Orion Browser
    Download and install the Orion browser on your iOS Device.
  2. Enable Firefox Extension Support IMPORTANT
    Open Orion → Tap “…” (bottom right) → Settings → Advanced → Turn ON “Firefox Extensions Support”
  3. Prepare the Extension
    Download the extension .zip file → Unzip so you have a normal folder.
  4. Move the Extension Folder
    Open Files app → Move unzipped folder into:
    Orion → Extensions
  5. Install the Extension in Orion
    Open Orion → Tap “…” → Extensions → Install from File → Open Orion → Extensions folder → Select the extension folder → Select / install manifest.json
  6. Done
    The extension should now be installed and active.
